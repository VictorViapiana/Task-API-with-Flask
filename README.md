# API de Tarefas com Flask

API REST desenvolvida em Python utilizando Flask, com operações completas de CRUD (Create, Read, Update, Delete).

## Funcionalidades

- Listar tarefas (GET)
- Criar tarefas (POST)
- Atualizar tarefas (PUT)
- Deletar tarefas (DELETE)

## Tecnologias

- Python
- Flask

## Como rodar

pip install flask
python app.py
